## Ex1

##### 源代码

```c
#include <stdio.h>

int main() {
    int a[10];
    int max = 0;

    for (int i = 0; i < 10; i++) {
        scanf("%d", &a[i]);
    }

    for (int i = 9; i >= 0; i--) {
        printf("%d ", a[i]);
        max = a[i] > a[max] ? i : max;
    }
    printf("\nthe index of max element in a is: %d\n", max);

    return 0;
}
```
